data.go.kr > 정보공유 > 개발자네트워크
header, nav, section, article, aside, footer 태그를 사용해 본다.

1. 상단  header의 내비게이션 항목을 만든다
- 데이터찾기, 국가데이터맵, 데이터요청, 데이터활용, 정보공유, 이용안내

2. 내비게이션 항목 중 정보공유를 실제 사이트처럼 표시한다.
- 배경색은 #0670b9
- 각 항목간에는 마지막항목을 제외하여 오른쪽 마진 100px 을 적용
- 정보공유 항목에는  active  클래스를 지정하여 아래쪽 선 표시와 글자색 적용

3. 실제 사이트의 내비게이션처럼 각 내비게이션 항목에 마우스 올렸을때의 처리를 적용

4. 테이블을 실제 사이트 페이지의 테이블 형태로 만든다.
- 제목줄의 위쪽선과 배경색을 적절한 색으로 적용
- 데이터 목록 중 제목 컬럼은 텍스트 왼쪽 정렬, 글자를 굵게 적용
- colgroup 의 각  col로 너비를 적절히 적용한다
- 실제 사이트처럼 제목에 마우스 올렸을때의 처리를 적용

5. 이용약관, 개인정보처리방침 등의 항목을  aside 안에 만들어
실제 사이트에서처럼 표시한다.
- 위쪽선을 적용
- 개인정보방침에 색상 지정
- 개인정보분야별책임자 : 최시복 를 제외한 각 항목에 마우스 올렸을때의 처리적용

6. footer 에 주소와 copyright 를 표시한다.
- 주소에 대해 address 태그를 적용하고 글자가 눕지않게 처리한다.
